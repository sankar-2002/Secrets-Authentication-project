Tech stacks used are -->

            # For Front End #

             * HTML - Skeleton Of The Project
             * CSS - Used For Styles
             * Bootstrap - Framework Used For Styling Webpages
             * JavaScript - Used For Implementing Logic and Reaction to Button Press

            # For Back End #

             * Node.js - Backend Server codes
             * Express.js - Framework for Node.js Which Creates server
             * Body-Parser - To parse The Forms
             * EJS(Embedded JavaScript) - Used When We Use JavaScript In HTML Files And Import Export Of partials
             * MongoDB - Database Used To Store Users Data
             * Mongoose - ODM(Object Data Modelling) Used For MongoDB
             * Md5 Package - Used For Password Hashing And Security

             * Partials Like Header And Footer includes Boiler Plate Codes

             Browser Used - Google Chrome
             Terminal Used - Hyper
             Robo3T - Used For Visualizing The Database
